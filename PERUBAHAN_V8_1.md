# Aiyone v8.1 Motion Polish

Perbaikan UI/UX kecil untuk mobile:

- Mengurangi efek kedip saat pindah menu.
- Menghapus animasi masuk halaman yang bikin tampilan terasa “membetulkan diri”.
- Menghapus efek biru/tap highlight bawaan browser di tombol dan opsi quiz.
- Mengurangi pulse shadow setelah auto-scroll.
- Cache PWA dinaikkan ke v8.1 supaya HP mengambil CSS/JS terbaru.
- Scroll halaman default dibuat instant, sementara auto-focus tertentu tetap smooth.

Catatan: setelah update di Vercel, uninstall/hapus PWA lama atau clear site data kalau HP masih menampilkan versi lama.
